--- title: "App Features" description: "Learn about the features Digital Trails provides to researchers." ---

Built by researchers for researchers

Digital Trails has been packed with features to support state of the art research. If you need a feature that doesn't currently exist, reach out. Our team can add features for you.
Get started

Native performance

Run studies natively on iOS and Android to get consistent performance for all participants.

Passive sensing

Collect data from on-device sensors according to custom sampling rates and rules.

Active sensing

Send surveys to participants according to either deterministic or stochastic schedules.

In-app Consent

Use any online platform to e-consent participants (e.g., Qualtrics, Docusign, or REDCap).

Privacy first

Protect privacy by storing personal identifiable information separately.

Data security

Secure data using encryption, both over the wire and in the cloud.
